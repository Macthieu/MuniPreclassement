# Security Policy

## Supported Versions

Le repository MuniPreclassement est maintenu sur la branche `main` et les tags de release officiels.

## Reporting a Vulnerability

- Utiliser en priorite les Security Advisories GitHub du depot.
- En attendant un canal dedie, ouvrir un rapport prive avec:
  - contexte,
  - impact,
  - etapes de reproduction,
  - version/tag concerne.

## Scope

Ce depot est actuellement au stade bootstrap. Le contrat CLI JSON est present, la logique metier reste a implementer.
